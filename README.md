# OpenCode Skill Creator

Exact Claude's skill creator but for OpenCode.

## What changes:
- All references from `Claude Code` / `Claude` were changed to `OpenCode` / `OpenCode.ai`.
- CLI commands modified from `claude` (like `claude -p`) to use the `opencode` CLI.
- Workspace configuration paths updated from `.claude/` to `.opencode/`.
- Environment variable checks (`CLAUDECODE`) updated to `OPENCODE`.
